# 請款單功能 — 進度記錄與待整合事項

> 最後更新：2026-07-21

## 目前狀態：先用單檔版

現階段先用**獨立單檔** `請款單產生器.html`（放在專案根目錄）。
- 雙擊即可用，不必跑 Python / 資料庫，資料存在瀏覽器 LocalStorage。
- 功能：輸入專案與合約總額 → 填各期比例自動算金額 → 產生請款單 → 列印 / 存 PDF → 「記錄為已送出業主」（日期＋方式）→ 明細有「狀態」欄（待收款／已開單／已收款）。
- 公司抬頭已預填：漣一設計有限公司、電話 0975-602-433、統編 83401776、Email laduree.design@gmail.com。
- 尚缺：匯款帳號（銀行／戶名／帳號），在檔案「公司資料 → 匯款資訊」填一次即可。

## 之後要做：串進整個營運系統（Flask `app/`）

主系統（`app/server.py` + `templates/`）已具備：案場、收款期別、`invoice_print.html`（同款版型）、公司資料設定頁。整合方向：

1. **資料合流**：把單檔版累積的資料（可用「匯出備份」出 JSON）匯入主系統 DB（`invoice_periods` / `company_info`）。
2. **搬功能到主系統**：
   - 在期別上加「已送出業主」記錄欄位（送出日期／方式），並在案場頁顯示。
   - 請款單頁加「✅ 記錄為已送出業主」按鈕（對應單檔版的 `markSent`）。
   - 明細加「狀態」欄與「本期已收款」綠色標示（單檔版已做，移植樣式即可）。
3. **公司資料**：主系統的 `company_info` 現有列需在 `/settings` 手動補一次（程式預設值只對全新資料庫生效）。

> 單檔版與主系統的請款單 CSS/版型一致，移植時可直接沿用。
